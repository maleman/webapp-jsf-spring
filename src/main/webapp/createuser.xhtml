Crear cuenta

El nombre de usuario debe:

  • Empezar con una letra
  • Seguido de letras o numeros o (_ -)
  • Debe tener una longitud minima de 3
  • Debe tener una longitud maxima de 10

La Contraseña debe:

  • Debe tener una letra minúscula, una letra mayúscula,
  • un número, un carácter (\!@#$%^*?)
  • Debe tener una longitud minima de 8
